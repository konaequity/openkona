About KONASH Arena

What is this?

KONASH Arena is a side-by-side comparison tool for search agents. Inspired by LMArena / Chatbot Arena, it lets you pit two retrieval-augmented reasoning agents against each other on the same question and vote on which one produces a better answer.

How it works

  • Enter a question and select two model presets
  • Both agents search the same corpus and reason through steps
  • Watch their retrieval, reasoning, and answer steps stream in real time
  • Vote on which response is better
  • Results feed into the leaderboard

Blind Mode

When enabled, model names are hidden and column assignment is randomized. Model identities are revealed only after you vote.

Integration

Built on KONASH's RolloutGenerator and MODEL_PRESETS. Each agent runs a multi-step search-reason-answer loop over the configured corpus.

Keyboard Shortcuts

  • Ctrl+Enter — Run comparison
  • 1 — Vote Left
  • 2 — Vote Right
  • 3 — Vote Tie
